 ## Description

An Web app for authentication for login using JWT and Passport.js


```bash
$ git clone https://github.com/pnidhi26/authentication-app
```

## Installation

```bash
$ cd authentication-app
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

Server:
http://localhost: 3000
